# `031` removeProperty

## 📝 Instrucciones:

1. Escribe una función llamada `removeProperty`. Dado un objeto y una key, `removeProperty` elimina la key de ese objeto dado.

## 📎 Ejemplo:

```Javascript
let obj = {
  name: 'Sam',
  age: 20
}
removeProperty(obj, 'name');
console.log(obj.name); // --> undefined
```
