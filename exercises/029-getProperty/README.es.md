# `029` getProperty

## 📝 Instrucciones:

1. Escribe una función llamada `getProperty`. Dado un objeto y una key, `getProperty` retorna el valor de la propiedad con esa key.

## 📎 Ejemplo:

```Javascript
let car = {
  model: 'Toyota'
};
let output = getProperty(car, 'model');
console.log(output); // --> 'Toyota'
```
