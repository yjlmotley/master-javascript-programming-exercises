# `029` getProperty

## 📝 Instructions: 

1. Write a function called `getProperty`. Given an object and a key, `getProperty` returns the value of the property at the given key. 

## 📎 Example:

```Javascript
let car = {
  model: 'Toyota'
};
let output = getProperty(car, 'model');
console.log(output); // --> 'Toyota'
```
